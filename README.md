# task-1-mars
for easy part i learnt the linux commands which include 
mkdir: to create directory
cd: to change the directory
touch: used to add files inside a directory
ls: to list the files inside a folder 
find . -iname: to find all the files with an extension 
cat> file: to display the text inside the file
grep: used to print all the lines in a file containing a KEYWORD
wc -l: to count the occurrence of a word
date: date and time is displayed 
top: shows the overall cpu usage
sudo shutdown -h time: commands the computer to shutdown after a given time

echo:used to display a text just like print function 
exit: exits the program
